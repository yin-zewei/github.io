FBI树

发表于 2020-02-12 | 分类于 题 | 阅读次数
字数统计 216 | 阅读时长 1

原题链接：传送门

思路：
递归回溯
如果l < r，那么用二分，返回左子树和右子树的值，处理后输出
如果l == r，根据数字0或1输出并返回字母

代码：

```c++

#include<cstdio>
const int maxn = 1<<10 + 2;
char arr[maxn];
char ef(int l,int r){
    if(l<r){
        int t = l + (r - l)/2;
        char a = ef(l,t);//左子树的返回值
        char b = ef(t+1,r);//右子树的返回值
        if(a == 'B'&& b=='B'){
            putchar('B');
            return 'B';
        }else if(a == 'I' && b =='I'){
            putchar('I');
            return 'I';
        }else {
            putchar('F');
            return 'F';
        }
    }
    //l == r时的处理
    if(arr[l] == '0'){
        putchar('B');
        return 'B';
    }else{
    putchar('I');
    return 'I';
    }
}
int main(){
    int n;
    scanf("%d",&n);
    scanf("%s",arr);
    ef(0,(1<<n) - 1);
    return 0;
}

求素数

发表于 2020-02-12 | 分类于 算法 | 阅读次数
字数统计 369 | 阅读时长 1

介绍：要求1-n中有多少个数是素数

初级方法

对于n,从i = 2到i = sqrt(n)，一个个的遍历，耗时很大，会超时

埃拉托斯特尼筛选法

复杂度位O(nloglogn)
设定一个数组f[n],f[i] = 1表示i为素数，vis[n],vis[i] = 1表示是否访问过
从i = 2到n遍历，未访问的设为1，然后j = i+i ，开始遍历每个i的倍数，直到n

欧拉筛选法

对埃拉托斯特尼筛选法进行了改进，因为埃拉托斯特尼筛选法会对一些重复项进行重复判断，例如2的倍数与3的倍数重合了，那么每次筛选的时候就只使用已经求过的素数进行筛选,即用一个数组vectorv记录已经求得的质因数

其他

Sundaram筛选法、Atkin筛选法

效率比较

耗时：Sundaram < 欧拉 = Atkin < 埃拉托斯特尼。

空间占用：Sundaram < 埃拉托斯特尼 = Atkin < 欧拉。

麦森数

发表于 2020-02-11 | 分类于 题 | 阅读次数
字数统计 341 | 阅读时长 1

原题链接：传送门

描述：给定一个数P，求2^p-1的位数和后500位数字，不足500位前面补零。

思路：
前一问，直接换成10的底，位数 = log(2)*p+1
后一问，采用快速幂算法，把指数p想象成二进制数字10101…，2^p = 2^(2^j)*2^(2^k)…

代码：
```c++

#include<cstdio>
#include<cmath>
#include<cstring>
using namespace std;
int ans[1005],f[1005],tmp[1005];
//用于计算二进制P奇数位时的乘法运算
void f1(){
    memset(tmp,0,sizeof(tmp));
    for(int i = 1;i <= 500;++i){
        for(int j = 1;j <= 500;++j){
            tmp[i + j - 1] += ans[i] * f[j];//直接计算某位上的值，随后进行进位
        }
    }
    for(int i = 1;i <= 500;++i){
        tmp[i + 1] += tmp[i] / 10;
        tmp[i] %= 10;
    }
    memcpy(ans,tmp,sizeof(tmp));
}
//用于每次对f的更新
void f2(){
    memset(tmp,0,sizeof(tmp));
    for(int i = 1;i <= 500;++i){
        for(int j = 1;j <= 500;++j){
            tmp[i + j - 1] += f[i] * f[j];
        }
    }
    for(int i = 1;i <= 500;++i){
        tmp[i + 1] += tmp[i] / 10;
        tmp[i] %= 10;
    }
    memcpy(f,tmp,sizeof(tmp));
}
int main(){
    int p;
    scanf("%d",&p);
    printf("%d\n",(int)(log10(2)*p+1));
    f[1] = 2,ans[1]  = 1;
    //核心快速幂代码
    while(p){
        if(p&1) f1();//计算末位
        p >>= 1;
        f2();//更新乘数2^k
    }
    ans[1]-=1;
    for(int i = 500;i >= 1;--i){
        if(i % 50 == 0 && i != 500)putchar('\n');
        printf("%d",ans[i]);
    }
    return 0;
}

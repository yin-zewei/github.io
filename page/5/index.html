FBI树

发表于 2020-02-12 | 分类于 算法

原题链接：传送门

思路：
递归回溯
如果l < r，那么用二分，返回左子树和右子树的值，处理后输出
如果l == r，根据数字0或1输出并返回字母

代码：

```c++

#include<cstdio>
const int maxn = 1<<10 + 2;
char arr[maxn];
char ef(int l,int r){
    if(l<r){
        int t = l + (r - l)/2;
        char a = ef(l,t);//左子树的返回值
        char b = ef(t+1,r);//右子树的返回值
        if(a == 'B'&& b=='B'){
            putchar('B');
            return 'B';
        }else if(a == 'I' && b =='I'){
            putchar('I');
            return 'I';
        }else {
            putchar('F');
            return 'F';
        }
    }
    //l == r时的处理
    if(arr[l] == '0'){
        putchar('B');
        return 'B';
    }else{
    putchar('I');
    return 'I';
    }
}
int main(){
    int n;
    scanf("%d",&n);
    scanf("%s",arr);
    ef(0,(1<<n) - 1);
    return 0;
}

单词接龙

发表于 2020-02-12 | 分类于 算法

原题链接：传送门

描述：单词接龙是一个与我们经常玩的成语接龙相类似的游戏，现在我们已知一组单词，且给定一个开头的字母，要求出以这个字母开头的最长的“龙”（每个单词都最多在“龙”中出现两次），在两个单词相连时，其重合部分合为一部分，例如 beastbeast和astonishastonish，如果接成一条龙则变为beastonishbeastonish，另外相邻的两部分不能存在包含关系，例如atat 和 atideatide 间不能相连。

思路：直接搜索

代码：
```c++

#include<cstdio>
#include<string>
#include<iostream>
int max(int a,int b){return a < b ? b :a;}
int min(int a,int b){return a < b ? a :b;}
int n,vis[25] = {0},ans = 0,len[25];
std::string s[25];
void dfs(std::string str){
    ans = max(ans,str.length());
    int tt = str.length();
    for(int i = 0;i < n;++i){
        if(vis[i] < 2){
            int t = min(tt,len[i]) - 1,flag = 0,j = 1;
            for(;j <= t;++j){
                if(s[i].substr(0,j)==str.substr(tt - j)){
                    flag = 1;
                    break;
                }
            }
            if(flag){
                ++vis[i];
                dfs(str + s[i].substr(j));
                --vis[i];
            }
        }
    }
}
int main(){
    scanf("%d",&n);
    for(int i = 0;i < n;++i){
        std::cin>>s[i];
        len[i] = s[i].length();
    }
    char start;
    std::cin >> start;
    for(int i = 0;i < n;++i){
        if(s[i][0] == start){
            ++vis[i];
            dfs(s[i]);
            --vis[i];
        }
    }
    printf("%d",ans);
    return 0;
}

求素数

发表于 2020-02-12 | 分类于 算法
  • 介绍：要求1-n中有多少个数是素数

    初级方法

    对于n,从i = 2到i = sqrt(n)，一个个的遍历，耗时很大，会超时

    埃拉托斯特尼筛选法

    复杂度位O(nloglogn)
    设定一个数组f[n],f[i] = 1表示i为素数，vis[n],vis[i] = 1表示是否访问过
    从i = 2到n遍历，未访问的设为1，然后j = i+i ，开始遍历每个i的倍数，直到n

欧拉筛选法

对埃拉托斯特尼筛选法进行了改进，因为埃拉托斯特尼筛选法会对一些重复项进行重复判断，例如2的倍数与3的倍数重合了，那么每次筛选的时候就只使用已经求过的素数进行筛选,即用一个数组vectorv记录已经求得的质因数

其他

Sundaram筛选法、Atkin筛选法

效率比较

耗时：Sundaram < 欧拉 = Atkin < 埃拉托斯特尼。

空间占用：Sundaram < 埃拉托斯特尼 = Atkin < 欧拉。

麦森数

发表于 2020-02-11 | 分类于 算法

原题链接：传送门

描述：给定一个数P，求2^p-1的位数和后500位数字，不足500位前面补零。

思路：
前一问，直接换成10的底，位数 = log(2)*p+1
后一问，采用快速幂算法，把指数p想象成二进制数字10101…，2^p = 2^(2^j)*2^(2^k)…

代码：
```c++

#include<cstdio>
#include<cmath>
#include<cstring>
using namespace std;
int ans[1005],f[1005],tmp[1005];
//用于计算二进制P奇数位时的乘法运算
void f1(){
    memset(tmp,0,sizeof(tmp));
    for(int i = 1;i <= 500;++i){
        for(int j = 1;j <= 500;++j){
            tmp[i + j - 1] += ans[i] * f[j];//直接计算某位上的值，随后进行进位
        }
    }
    for(int i = 1;i <= 500;++i){
        tmp[i + 1] += tmp[i] / 10;
        tmp[i] %= 10;
    }
    memcpy(ans,tmp,sizeof(tmp));
}
//用于每次对f的更新
void f2(){
    memset(tmp,0,sizeof(tmp));
    for(int i = 1;i <= 500;++i){
        for(int j = 1;j <= 500;++j){
            tmp[i + j - 1] += f[i] * f[j];
        }
    }
    for(int i = 1;i <= 500;++i){
        tmp[i + 1] += tmp[i] / 10;
        tmp[i] %= 10;
    }
    memcpy(f,tmp,sizeof(tmp));
}
int main(){
    int p;
    scanf("%d",&p);
    printf("%d\n",(int)(log10(2)*p+1));
    f[1] = 2,ans[1]  = 1;
    //核心快速幂代码
    while(p){
        if(p&1) f1();//计算末位
        p >>= 1;
        f2();//更新乘数2^k
    }
    ans[1]-=1;
    for(int i = 500;i >= 1;--i){
        if(i % 50 == 0 && i != 500)putchar('\n');
        printf("%d",ans[i]);
    }
    return 0;
}

采药

发表于 2020-02-11 | 分类于 算法

原题链接：[传送门][https://www.luogu.com.cn/problem/P1048]

描述：时间T内，M种药，每种药需要不同的时间，并含有不同的价值，问能获得的最大价值是多少？

思路：简单的tp问题，作一个dp[M][T]数组，表示T时间内前M的药物能获得的最大价值，然后递增即可。

传递方程：dp[i][j] = max(dp[i - 1][j - a[i]] + b[i],dp[i - 1][j]);

代码：
```c++

#include<cstdio>
#include<cstring>
using namespace std;
int max(int x,int y){return x < y ? y : x;}
int a[105],b[105],dp[105][1005] = {0},T,M;
int main()
{
    scanf("%d %d",&T,&M);
    for(int i = 1;i <= M;++i) scanf("%d %d",&a[i],&b[i]);
    a[0] = b[0] = 0;
    for(int i = 1;i <= M;++i){
        for(int j = 0;j <= T;++j){
            if(j >= a[i])
                dp[i][j] = max(dp[i - 1][j - a[i]] + b[i],dp[i - 1][j]);
            else 
                dp[i][j] = dp[i - 1][j];
        }
    }
    printf("%d",dp[M][T]);
    return 0;
}

南蛮图腾

发表于 2020-02-10 | 分类于 算法

原题链接：传送门

描述：给出一个数字，输出一定规律的图腾

思考：直接初始化一个足够大的二维数组，做盛放字符的容器，然后一步步将原规律复制到数组中。

坑点：数组必须在主函数内部进行初始化，否则洛谷会报错，但本地运行良好。

代码：
```c++

#include<cstdio>
using namespace std;
char arr[1024][2048];
int main(){
    int n;
    scanf("%d",&n);
    for(int i = 0;i < 1024;++i){
        for(int j = 0;j < 2048;++j)
        arr[i][j] = ' ';
    }
    arr[0][0] = arr[1][1] = '/';
    arr[0][1] = arr[0][2] = '_';
    arr[0][3] = arr[1][2] = '\\';
    int l = 4, h = 2;//where next group start copying
    while(--n){
        for(int i = 0;i < h;++i)
        for(int j = 0;j < l;++j){
            arr[i + h][j + h] = arr[i][j + l] = arr[i][j];
        }
        l <<= 1;
        h <<= 1;
    }
    for(int i = h - 1;i >= 0;--i){
        for(int j = 0;j < l;++j){
            putchar(arr[i][j]);
        }
        putchar('\n');
    }
    return 0;
}

逆序对

发表于 2020-02-09 | 分类于 算法

描述：给定一个数字序列，求满足i < j && arr[i] > arr[j]的个数

原题链接：传送门

思路：使用归并排序，根据排序原理，排序中间添加一行计数器代码，每次当左边数组的数字大于右边数组的数字的时候，说明左边数组存在m - i + 1个数(i往右边数直到中数m)大于右边数组的数，那么计数器加上m-i+1，最后计数器的数就是答案。

代码：

复杂度为O(nlogn)
```c++

#include<cstdio>
using namespace std;
#define ll long long
const int maxn = 5e5 + 5;
int arr[maxn],t[maxn]; 
ll ans = 0;
void merge(int l,int r,int m);
void msort(int l,int r){
    if(l < r){
        int m = l + (r - l)/2;
        msort(l,m);
        msort(m+1,r);
        merge(l,r,m);
    }
}
void merge(int l,int r,int m){
    int i = l,j = m + 1,id = l;
    while(i <= m && j <= r){
        if(arr[i] <= arr[j]){
            t[id++] = arr[i++];
        }else{
            ans += (ll)m - i + 1;
            t[id++] = arr[j++];
        }
    }
    while(i <= m) t[id++] = arr[i++];
    while(j <= r) t[id++] = arr[j++];
    while(l<=r) arr[l] = t[l],++l;
}
inline int read(){
    int p = 0,flag = 1;
    char c = getchar();
    while(c < '0' || c > '9'){if(c == '-') flag = -1;c = getchar();}
    while(c >= '0' && c <= '9') p = p*10 + c - '0',c = getchar();
    return p*flag;
}
int main(){
    int n = read();
    for(int i = 0;i < n;++i)arr[i] = read();
    msort(0,n - 1);
    printf("%lld",ans);
    return 0;
}

c++快读整数

发表于 2020-02-09 | 分类于 算法

一般来讲，c++读入一个整数可以使用cin或者scanf()，而且scanf()函数会快一点，然而在大批整数的读取过程中，可以使用getchar()函数读取，速度更快，更省时间，因为scanf()需要缓冲，需要解析字符串，而getchar()直接读入一个字符，同理，putchar()也比printf()快一点。
```c++
int read(){
int res = 0;
char c = getchar();
while(c < ‘0’ || c > ‘9’) c = getchar();
while(c >= ‘0’ && c <= ‘9’) res = res*10 + c - 48,c = getchar();
return res;
}

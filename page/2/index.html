欧拉函数

  • 定义：φ(n)即欧拉函数，为1-n中与n互质的数的个数
  • 互质：a与b互质即a与b只有1一个公因数，特殊地，1与1互质，其他任何数与它本身不互质

倍增LCA

发表于 2020-05-03 | 分类于 算法 | 阅读次数
字数统计 428 | 阅读时长 2

LCA是指求最近公共祖先的算法，如果使用暴力，那就是从下至上，但肯定会爆RE
使用倍增优化后，复杂度nlogn
步骤：

  • 初始化
    • 数据输入
    • lg优化(初始化lg[i]=log2(i)+1)
    • dfs初始化fa[i][j]=i节点的2^j祖先
  • LCA
    • 将a,b的深度调为一致
    • 如果相同，返回该祖先
    • 如果不同
      • i = lg(a)-1 -> 0,i 为 倍增的倍数
      • 如果fa[a][i] != fa[b][i]向上调节，逐渐到LCA的儿子节点

最小生成树

发表于 2020-05-02 | 分类于 算法 | 阅读次数
字数统计 647 | 阅读时长 3

以下介绍两种：Prim和Kruskal

Prim
Prim适用于稠密图，堆优化后复杂度VlogV
主要步骤：

* 随意取一条边
* 将这条边相邻的边加进来
* 取一个最小的边
* 直到有n-1条边

Kruskal
Kruskal适用于稀疏图
主要步骤：

* 将边排序
* 取最短边
* 查看是否连通
* 不连通就加进来，连通就跳过
* 继续取最短边，直到达到n - 1条

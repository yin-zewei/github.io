南蛮图腾

Posted on 2020-02-10 | In 算法

原题链接：传送门

描述：给出一个数字，输出一定规律的图腾

思考：直接初始化一个足够大的二维数组，做盛放字符的容器，然后一步步将原规律复制到数组中。

坑点：数组必须在主函数内部进行初始化，否则洛谷会报错，但本地运行良好。

代码：

#include<cstdio>
using namespace std;
char arr[1024][2048];
int main(){
    int n;
    scanf("%d",&n);
    for(int i = 0;i < 1024;++i){
        for(int j = 0;j < 2048;++j)
        arr[i][j] = ' ';
    }
    arr[0][0] = arr[1][1] = '/';
    arr[0][1] = arr[0][2] = '_';
    arr[0][3] = arr[1][2] = '\\';
    int l = 4, h = 2;//where next group start copying
    while(--n){
        for(int i = 0;i < h;++i)
        for(int j = 0;j < l;++j){
            arr[i + h][j + h] = arr[i][j + l] = arr[i][j];
        }
        l <<= 1;
        h <<= 1;
    }
    for(int i = h - 1;i >= 0;--i){
        for(int j = 0;j < l;++j){
            putchar(arr[i][j]);
        }
        putchar('\n');
    }
    return 0;
}

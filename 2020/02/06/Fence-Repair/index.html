Fence Repair

发表于 2020-02-06 | 分类于 算法

来源

描述：栅栏修补，一共需要n个木条，长度不一，现要求从一个刚好够所有长度的木条上锯开，怎么锯划算，每次的花费是木条长度，如锯开21的木条，需要花费21，求最小花费。

思路：使得较长的木条使用的次数最少即可，那么从短的开始锯，每次计算两根，然后他们的和入队，重新排列。

注意：答案可能溢出，所以用long long int存储。
```c++

#include<cstdio>
#include<algorithm>
#include<queue>
using namespace std;
const int maxn = 20005;
int n,fence[maxn],t;
long long sum = 0;
int main(){
    scanf("%d",&n);
    //优先队列的使用
    priority_queue<int,vector<int>,greater<int>>q;
    for(int i = 0;i < n;++i){
        scanf("%d",&t);
        q.push(t);
    }
    while(q.size() > 1){
        //每次将最短的两根拿出来算
        int a = q.top();
        q.pop();
        int b = q.top();
        q.pop();
        sum += a + b;
        q.push(a+b);
    }
    printf("%lld",sum);
    return 0; 
}

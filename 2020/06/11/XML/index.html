XML

发表于 2020-06-11 | 分类于 javaweb | 阅读次数
字数统计 825 | 阅读时长 3
  • 概念：Extensible Markup Language 可扩展标记语言，全部标签都是自定义的
  • 功能：存储数据
    • 配置文件
    • 在网络中传输
  • 与html的区别
    • xml自定义，html是预定义
    • xml语法严格，html语法松散
    • xml存储数据，html展示数据
  • 语法
    • 基本语法
      • 后缀是.xml
      • xml第一行必须是文档声明
      • xml有且仅有一个跟标签
      • 属性必须用引号引起来
      • 标签必须正确关闭
      • 标签区分大小写
    • 组成
      • 文档声明
        • 格式：<?xml version=’1.0’ encoding=’utf-8’?>
        • 属性列表：
          • version:版本号声明
          • encoding:当前文档编码方式，默认ISO-8859-1
          • standalone:是否独立，yes or no
      • 指令：控制样式，现在基本废弃
      • 标签：名称自定义
        • 名称有限制：除了和java变量的限制外，还不能已xml的任意形式开始
      • 属性：id值唯一
      • 文本：
        • CDATA区：在该区域内的数据会原样显示
          • 格式：< ![CDATA[数据]] >
  • 解析XML
    • 谁编写？—软件使用者编写
    • 谁解析？—软件
    • 特殊之处
      • 软件为半成品软件
      • 使用者为程序员
    • 约束文档：规定xml的编写规则
    • 对程序员要求：会简单引入约束文档和读懂约束文档
    • 分类
      • DTD
        • 内部DTD：约束规则在xml文档中
        • 外部DTD：约束规则在xml文档外
          • 本地：<!DOCTYPE 根标签名 SYSTEM “dtd文件地址”>
          • 网络：<!DOCTYPE 根标签名 PUBLIC “dtd文件名” “dtd文件地址”>
      • Scheme
        • 引入
          1. 填写xml文档的根元素
          2. 引入xsi前缀. xmlns:xsi=”http://www.w3.org/2001/XMLSchema-instance“
          3. 引入xsd文件命名空间. xsi:schemaLocation=”http://www.itcast.cn/xml student.xsd”
          4. 为每一个xsd约束声明一个前缀,作为标识 xmlns=”http://www.itcast.cn/xml“
    • 解析与写入
      • 解析：将xml文档中的数据读取到内存中
      • 写入：将内存中的数据保存到xml文档中
      • 解析的方法
        • DOM：将标记语言文档一次性加载进内存中，在内存中形成一颗DOM树
          • 优点：操作方便，可以对文档进行CRUD的所有操作
          • 缺点：占内存
        • SAX：逐行读取，基于事件驱动
          • 优点：不占内存
          • 缺点：只能读取，不能增删改
      • 常见的解析器
        • JAXP：支持dom和sax两种思想
        • DOM4J：优秀的DOM解析器，常用语pc端
        • Jsoup：dom解析器，是一款java的html解析器，可直接解析url或文本，可通过dom,css,jquery等方法取出操作数据
        • PULL：android内置sax解析器
      • 解析器Jsoup的使用
        • 导入jar包
        • 获取Document对象
        • 获取对应的标签Element对象
        • 获取数据
        • 例
    • 对象的使用
      • Jsoup：工具类，用于解析html或xml文档，返回Document
        • parse(File in,String str)，通过文件和编码方法
        • parse(String str)，str为字符串
        • parse(URL url,int timeoutMillis)通过网络路径
      • Document：文档对象，代表内存中的Dom树
      • Elements：文档Element对象的集合，可以当做ArrayList来使用
      • Element：元素对象
      • Node：节点对象
